# Ankh-Morpork: Guild Wars

A Discworld-inspired board game of influence, corruption, and mild chaos.

---

## 📁 What's Here

This folder contains a complete playable prototype:

### Core Files
- **`ankh-morpork-guild-wars.md`** - Full game concept document (read this first)
- **`rules/quick-start.md`** - Simplified rules to start playing immediately
- **`rules/guild-reference.md`** - Guild abilities and strategies

### Game Components
- **`cards/event-cards.md`** - 40 Event cards (Patrician Decrees, Watch raids, chaos)
- **`cards/contract-cards.md`** - 30 Contract cards (objectives and rewards)
- **`board/district-map.md`** - Ankh-Morpork city map with 8 districts

---

## 🎮 How to Play (Quick Version)

1. **Setup:** Choose a guild, place 2 starting influence tokens
2. **Goal:** Control 5 districts, reach 50 gold, OR complete 3 major contracts
3. **Each Turn:**
   - Collect income from your districts
   - Take 3 actions (place influence, complete contracts, use abilities)
   - Roll for Watch raids
   - Draw and resolve an Event card
4. **Win:** First to achieve a victory condition

**Full rules:** See `rules/quick-start.md`

## 🖨️ How to Prototype

### Option 1: Print-and-Play (Physical)
1. Print the district map (`board/district-map.md`) on A3 paper
2. Print Event cards (40) and Contract cards (30) on cardstock
3. Use poker chips or coins as influence tokens (5 colors for 5 guilds)
4. Use real coins for gold, other tokens for favors
5. 3 six-sided dice

### Option 2: Tabletop Simulator (Digital)
- Import card images and board layout
- Use built-in tokens/pieces
- Fully scriptable for automation

### Option 3: Simple Playtest (Minimal)
- Use notebook paper for districts (just write district names)
- Playing cards as Event/Contract proxies (write on them in pencil)
- Pennies as influence tokens (different colors marked with tape)

## 🎲 Playtesting Goals

- **Pacing:** Does the game end in 45-90 min?

## 🔥 Design Philosophy

**Core Tenets:**
1. **Embrace Discworld's Chaos** - Nothing works perfectly, bureaucracy is absurd
2. **Player Agency Matters** - Decisions feel meaningful even when events screw you
3. **Negotiation > Conflict** - Talking your way out is very Discworld
4. **Flavor > Mechanics** - If it's funny/thematic, keep it even if slightly unbalanced

**Inspirations:**
- Cosmic Encounter (negotiation, chaos)
- Root (asymmetric factions)
- Ticket to Ride (area control simplicity)
- Discworld board game by Martin Wallace (theme)

## 📚 Discworld References

Every card, mechanic, and district draws from Terry Pratchett's books:
- The Watch series (Guards! Guards!, Night Watch, etc.)
- Going Postal / Making Money
- The Truth
- Thud!
- And many others

**Key Themes:**
- The city as a character
- Lawful corruption ("It's not a crime if there's a guild for it")
- Commander Vimes' cynical idealism
- Lord Vetinari's omniscient control
- The Luggage (always hungry, always loyal)
